Where can I find C1 forms to try this out?

You can download some example forms. Otherwise, all C1 forms for the national elections can be found at the KPU site.

Why do we need kawal C1 if we have KPU and kawal pemilu?

This application is not meant duplicate the work of KPU and kawal pemilu but to complement and to make their work easier. The idea is for this application to improve both the quality and speed of the manual verification.

How does this application recognize the handwritten numbers?

The application uses a neural network , which was trained on a set of 70.000 sample numbers. Training is done by feeding the network with a lot of examples of different categories of digits (one through nine). After having seen 6.000 different 4's for example, in all its variations, the network kind of knows what a 4 looks like and can recognize it with an accuracy of better than 99%.

About half of the numbers in the training set were manually selected by volunteers from C1 forms. The other half was acquired by exploiting the outcome of kawal pemilu crowd-sourcing initiative. This made it possible to pre-classify the digits which greatly sped up the process

The video on the right displays all numbers that are part of the training set. Initially we used the MNIST set. The main reason we couldn't get accurate enough results with this set is that it doesn't contain crosses, which the C1 forms do. Another reason is that it contains American handwriting, which seems to be subtly different from Indonesian handwriting.

What is the accuracy of this application?

This is something we cannot give reliable numbers on yet. This website should be seen as a proof of concept. However, since all C1 forms and both the official KPU and kawal pemilu resuls are openly available, it will be relatively easy to test and improve the accuracy. This is our top priority now.

Does this application only work for C1 forms of the 2014 presidential election?

At the moment yes. But it will not be difficult to configure it for other forms like the ones that are used for local elections for example.

Who can use this application?

This application is completely open source and can be integrated to any existing software used in the election process.

Isn't human verification more reliable than computer verification?

Yes, in most cases. This software serves as a tool to improve human verification.

What's next for this application?

These are some of the thing we would like to do in the near future:

  • Add support for different kinds of forms. For example, the ones that contain information about the number of people eligble to vote or forms for local elections
  • Make the datasets that are used by this application available to the machine learning community.

Which technologies do you use?

  • The website was built using AngularJS.
  • The webservices use the Django library.
  • The image registration was written in Python, using various image processing libraries like OpenCV, scikit-image and scipy.
  • The number classification is done use using a convolutional neural network that was trained on a digit database semi-automatically extracted form the C1 forms.

Aplikasi ini tidak dimaksudkan untuk mengulangi pekerjaan KPU atau yang sudah dikerjakan oleh kawal pemilu melainkan untuk melengkapi dan memudahkan pekerjaan mereka. Kami membuat aplikasi ini untuk bisa meningkatkan kwalitas dan kecepatan verifikasi secara manual.

Bagaimana aplikasi ini bisa mengenali angka-angka tulisan tangan?

Aplikasi ini menggunakan neural network , yang dilatih dengan menggunakan satu set sampel angka sebanyak 70.000 sampel angka. Latihan dilakukan dengan cara memasukkan sampel angka-angka (angka satu sampai dengan sembilan) yang ditulis tangan ke dalam komputer. Setelah mendapat misalnya sekitar 6.000 sampel angka 4, dengan tulisan tangan yang berbeda-beda, komputer akan bisa mengetahui seperti apa bentuk angka 4 dan bisa mengenalinya dengan tingkat akurasi lebih dari 99%.

Sekitar setengah dari sampel angka-angka yang digunakan untuk melatih komputer seperti dijelaskan di atas, diambil secara manual oleh beberapa sukarelawan dari C1 forms. Setengah lainnya diperoleh dengan memanfaatkan hasil dari inisiatif crowd-sourcing kawal pemilu. Dengan menggunakan data-data dari kawal pemilu, prosesnya menjadi lebih cepat karena tidak perlu mengambil data secara manual lagi.

Video di sebelah ini menampilkan semua sampel angka-angka yang digunakan untuk melatih komputer. Awalnya kami menggunakan data set dari MNIST. Namun sampel ini kurang bagus dan membuat aplikasi komputer kurang akurat karena sampel ini tidak mempunyai “x” yang digunakan di formulir C1. Selain itu sampel ini merupakan tulisan tangan orang Amerika yang sedikit berbeda dengan tulisan tangan orang Indonesia.

Bukankah verifikasi oleh manusia lebih dapat dipercaya daripada komputer?
